### About the Pokemon API

<a class="github-button" href="https://github.com/darenliang/pokemonapi" data-size="large" aria-label="Visit darenliang/pokemonapi on GitHub">Visit on GitHub</a>

PokemonAPI is a project that I am working on to learn how to develop web apps. From the short amount of time I am working on this project, I have found that it is truly easy to create fast and efficient APIs with Go. The gorilla/mux package makes managing API paths easy, and running the server uses surprisingly low resources.

As I work on the project, I would like to make a frontend web app along with the API.

Stay tuned for more information!
