### About the Go Jikan API Wrapper

<a class="github-button" href="https://github.com/darenliang/jikan-go" data-size="large" aria-label="Visit darenliang/jikan-go on GitHub">Visit on GitHub</a>

The Go Jikan API wrapper makes it easy to access the [Jikan REST API](https://jikan.moe/) which parses data from MyAnimeList, a popular anime manga social networking site.

Huge props to the Go team for including a http package in the standard Go library. Because of this, jikan-go does not need additional packages.

You can install the package with `go get github.com/darenliang/jikan-go` and find the documentation at [godoc](https://godoc.org/github.com/darenliang/jikan-go).
